Quickly edit your ACF content without having to leave your page.

== Description ==
With Quick Edit ACF Content you can edit your ACF content without having to leave the page you're editing.
All ACF fields are supported - including ACF Pro fields.

== Installation ==
= Installation =
1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. View a page in the frontend and click "Quick ACF" in the top black admin-bar and start editing.

== Required plugins ==
You'll need to have ACF 6.0+ (free or PRO) installed for this plugin to work.

== Frequently Asked Questions ==
= Does this plugin require to have ACF installed? =
You can install the plugin without having ACF 6.0+ installed but you won't be able to use it.

= What ACF fields can i edit? =
All of them.
